# planV9.9 — Thought Inspector Beginner UX + Versioned Preset Code

This plan upgrades MIND’s Thought Inspector to a **Beginner-first** UX with:
- Beginner knobs (role-aware, voice-type-aware)
- Advanced expansions of each beginner knob
- A **versioned, reversible Preset Code** (settings ⇄ code) that also drives deterministic generation
- A **Rebuild/Recompile** loop so edits apply to what plays

**Playback order remains graph-driven** (connections define order). “Song parts” are **roles** attached to Thoughts, not a forced timeline.
