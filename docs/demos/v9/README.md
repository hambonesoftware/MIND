# V9 Acceptance Demos

These demo graphs showcase the V9 flow runtime and Thought authoring model.

## How to run
1. Start the server: `python run.py`.
2. In the UI transport bar, select a **V9 Demo** and click **Load V9 Demo**.
3. Press **Play** to hear the demo.

## Demos
- **Moonlight Opening Loop**: Start → Moonlight Thought → Counter → Switch (loop N times, exit to outro).
- **Parallel Fan-out**: Start → Thought A + Thought B (parallel).
- **Join Barrier**: Start fan-out → Thought A + Thought B → Join → Thought C.

Each demo includes a description, expected behavior, and success criteria in its JSON.
